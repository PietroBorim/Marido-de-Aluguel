# ServiçoJusto — Marido de Aluguel

## Sobre o projeto

O ServiçoJusto é uma landing page funcional criada para apresentar uma solução digital de comparação de prestadores de serviços locais.

A proposta do projeto é ajudar usuários a encontrar, filtrar e comparar profissionais de áreas como elétrica, hidráulica, pintura e jardinagem, utilizando critérios como avaliação, preço por hora e distância.

O projeto foi desenvolvido como um MVP, ou seja, uma primeira versão funcional para demonstrar a ideia principal da solução.

## Link da página publicada

https://pietroborim.github.io/Marido-de-Aluguel/

## Funcionalidades

* Home Page com apresentação do projeto;
* Campo de busca inicial;
* Listagem de prestadores em destaque;
* Página de busca por nome ou categoria;
* Filtro por categoria;
* Ordenação por avaliação, menor preço e distância;
* Seleção de até 3 prestadores para comparação;
* Página de comparação entre prestadores;
* Modal de cadastro;
* Notificações visuais para ações do usuário.

## Tecnologias utilizadas

* HTML5;
* CSS3;
* JavaScript;
* Bootstrap;
* Google Fonts.

## Estrutura dos arquivos

* `index.html`: estrutura principal da página;
* `style.css`: estilos visuais, cores, cards, botões, modal e responsividade;
* `app.js`: funcionalidades interativas, como busca, filtros, ordenação, comparação e notificações.
